# Mora Jai Box solver

![demo.gif](assets/demo.gif)

This is the script to solve Mora Jai box (a puzzle toy in the video game [Blue Prince](https://store.steampowered.com/app/1569580/Blue_Prince/))

It is not the fast, nor the most easy to use way to solve Mora Jai box.  
It just why not make my own one? And it probably solve the box faster than me.  
Also I want to make something this great game and my journey. 

If you really need some help on the boxes, I recommend [Joric's box solver](https://joric.github.io/blueprince/)
